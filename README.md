Este projeto foi desenvolvido como atividade prática da disciplina de Internet das Coisas em um Mundo Conectado, do curso Análise e Desenvolvimentos de Sistemas da PUCPR.
Com o objetivo principal de ser um sistema de estufa, monitorando a umidade e temperatura, alertando caso ultrapasse as condições.

Desenvolvi utilizando o site do Wokwi, tanto para criar o código quanto para a simulação do hardware físico.
Utilizado em conjunto com o MQTTX, para realizar a conexão com o Blynk, atualizando as informações no painel web e aplicativo móvel.
Bibliotecas foram disponibilizadas pelo professor.

Status: Projeto concluído e utilizado somente para fins acadêmicos.
Data: 27/09/2025.
